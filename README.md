> MIRA: CS:GO Legacy Enhancement Suite

>**MIRA** — это комплексная модификация для Counter-Strike: Global Offensive, направленная на восстановление и расширение функционала игры, который был доступен до официального закрытия/перехода на новые версии. Проект возвращает привычные возможности и добавляет современные инструменты для работы с контентом.

>**Основные компоненты**

1.  **MIRA Core**: Ядро модификации, восстанавливающее сетевые и интерфейсные возможности "классической" CS:GO.
2.  **MIG-System**: Специализированный модуль для интеграции и управления внешними ресурсами.
3.  **Addons Support**: Система поддержки пользовательских дополнений, расширяющих базовый функционал мода.

>**Ключевые особенности**

*   **Legacy Restoration**: Возвращение функционала игры в состояние "до закрытия".
*   **Real-time News System**: Динамическая система новостей внутри Panorama UI (реализована через эффективную доставку данных).
*   **MIG Integration**: Улучшенная работа с моделями и ресурсами.
*   **Modular Architecture**: Легкая установка и удаление аддонов без повреждения файлов игры.

>**Установка**

1. Скачайте последнюю версию из раздела [Releases].
2. Распакуйте содержимое архива в корневую папку игры (там, где находится `csgo.exe`).
3. Для установки аддонов поместите файлы `.mira` (или соответствующие папки) в директорию `csgo/mira_addons/`.

>**Структура проекта**

*   `/core` — основные файлы модификации.
*   `/mig` — система управления мигами.
*   `/addons` — примеры и база для создания собственных дополнений.
