># MIRA LOADER 🌀

![MIRA LOADER Preview](https://raw.githubusercontent.com/K3rrr1337/MIRA-files/main/MIRALoader.png)

>**MIRA LOADER** — это легковесный и интуитивно понятный инструмент для автоматизированной установки модификаций в CS:GO. Проект создан для обеспечения максимально быстрого и безопасного процесса обновления игровых файлов.

>## ❔ Особенности

*   **Простота**: Установка в один клик.
*   **Скорость**: Загрузка ресурсов через Github.
*   **Безопасность**: Полностью открытый исходный код.
*   **Язык**: Python + C# и .NET Интеграция

>## ☑ Установка

1.  Перейдите в раздел [Releases](https://github.com/K3rrr1337/MIRA-files/releases).
2.  Скачайте актуальный архив `dist.rar`.
3.  Запустите лоадер, укажите путь к игре и нажмите кнопку **Установить**.
4.  После завершения (100%) запустите `migi.exe` и нажмите UPDATE BUILD!.
5.  Затем зайдите в параметры запуска и укажите `-game migi/game`
